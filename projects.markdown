### Project 1: Spellwalk(WIP)
    This project is build off of python and pygame, and is also my first experience
    utilizing pygame for game development. Spellwalk is a top-down vampire survivor inspired
    game where the player must survive waves of enemies using various spells and abilities.
    (Currrently a work in progress as I continue to add more features and polish the game.)

- **Tech Stack**: Python, Pygame
- **Key Features**: Top-down vampire survivor inspired gameplay, wave-based enemy survival, various spells and abilities
- **Links**: https://github.com/anthonywwalter-beep/spellwalk1.0

### Project 2: Planner
    An experiment I decided to do in a single sitting after I realized I forgot
    to buy a planner for the school year. This is a simple web application built with
    HTML, CSS, and Javascript that allows users to create, view, and manage their daily
    tasks and events in a clean and organized interface. There are known bugs but overall
    it functions as intended, and was one off project to practice my web development skills.

- **Tech Stack**: HTML, CSS, Javascript
- **Key Features**: Task and event management, clean and organized interface
- **Links**: https://github.com/anthonywwalter-beep/planner

### Project 3: Word Wizards
    A Hackathon project created within 24 hours with a group of 4 for Quackhacks Fall 2025.
    Word Wizards is a typing game where players can compete against each other in real-time to see who can type
    the fastest and most accurately. Built with Javascript, HTML, and CSS for the frontend and Render and socket.io for the backend.
    I was responsible for the HTML and button functionality as well as some of the CSS styling, UI desgin.

- **Tech Stack**: Javascript, HTML, CSS, Render, socket.io
- **Key Features**: Real-time multiplayer typing competition, accurate and fast typing challenge
- **Links**: https://github.com/khozii/word-wizards
